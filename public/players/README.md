# Avatares de jugadores

Pon aquí las fotos de perfil de los jugadores con estos nombres exactos:

- `guille.jpg` (o .png / .webp)
- `ana.jpg`
- `edu.jpg`

Se recomienda:
- Formato cuadrado (1:1), ej. 256×256 o 512×512 px.
- Bajo peso: <100 KB cada una si es posible.

Si no hay imagen, la app muestra automáticamente un círculo de color como avatar.
